Change Data & Customize Your Web Template
( must do it before submitting the assignment )

Update Profile Image

open the folder images

replace the image "myPic.jpg" with your profile image

Note: make sure to rename your image to myPic.jpg in order to make it work

Update Your Info Data

open the file index.html in Notepad or any other text editor

search for the Header section

update your info

Update About Section

open the file index.html in Notepad or any other text editor

search for About me in the file

update the data and save

Update Other Sections

similarly you can update other sections

open the file index.html in Notepad or any other text editor

search for the elements view resume, view profile, view video, view gallery and blog in the file

update the data and save

Change Colour And Font

open the file css\style.css in Notepad or any other text editor

search for root in the file

you can change the rgb() value of these variables to change the colour scheme of the website

also, you may change the --font-family variable to change the font of the website (https://fonts.google.com/)
